Work

Here are some of the projects developed during my full stack webdevelopment course with CareerFoundry

Visit my GitHub page to see more info on each project in the ReadMe files.


Pokedex-App

This is a Pokedex application that gets data from an external API, pokeapi.

Features:

  • Fetches data from an external public API
  • A button for all original 150 Pokemon
  • Each button opens a modal for the selected Pokemon, offering details and an image
  • A search bar that allows you to search by name

The task for this project was to learn about the DOM manipulation. It was also an introduction to JavaScript, jQuery, and styling with Bootstrap.

Pokedéx hosted on GitHub-pages
Pokedéx repo on GitHub

Meet-App

A meet up app that fetches data from Google Calendar API

The task for this project was to learn about test-driven-development, serverless functions, OAuth, and Progressive Web Application's. We also touched on creating charts and monitoring performance.

Features:

  • Search by city name
  • Show and hide event details
  • Change the number of events shown
  • A chart showing the number of upcoming events in each city
  • A pie chart that shows the popularity of a specific event

This application was built with CreateReactApp and was tested using test driven development.

Tech used for this project was React, CSS, Jest, AWS lambda, OAuth, Axios, Recharts, and Atatus

Meet-App hosted on GitHub-pages
Meet-App repo on GitHub

Chat-App

A chat application for mobile devices running iOS and Android, created using React Native.

The task for this project was to learn how to develop and test cross-platform native apps.

Features:

  • Select username
  • Select your background color
  • Send and receive messages
  • Share pictures from your camera roll
  • Take a new photo to share
  • Share location
  • See messages while offline

Tech used for this project are JavaScript, React Native, Expo, Google Firebase.

Movie-API

This is an API created to store movie data.

Tech used for this project were Javascript, node.js (Express, Morgan, bodyParser, mongoose, CORS, Passport), DB on nMongoDB Atlas, Postman, and Heroku.
